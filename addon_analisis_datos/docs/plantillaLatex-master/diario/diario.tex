\chapter{Diario de desarrollo}

\section{Introducción}

En este anexo se recoge un diario de desarrollo del proyecto donde se documentan
las diferentes pruebas realizadas, los problemas encontrados y las soluciones
aplicadas durante la implementación del sistema.

Además de los cambios registrados en el sistema de control de versiones,
también se incluyen anotaciones correspondientes a días en los que se realizaron
pruebas, análisis o planificación del trabajo aunque no se generaran commits.
